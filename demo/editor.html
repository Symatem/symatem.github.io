Symatem Ontology Editor

What you can see below is an ontology engine written in C++17 which is part of a bigger project: Symatem, compiled to a binary WebAssembly file (88 KB) using Clang 4.0 and loaded into the browser. There some ES6 code, which handles the user interaction and the rendering, is connected to it via these two files: C++ side, JS side.

    (Car; Has (IsA Group; Elements Wheel; Count 4;);)


Usage

  • Explore:
    • Drag and drop a symbol (box) around to reposition it
    • Click on a green or blue socket (circle) to explore the directly connected symbol
    • Click on a socket a second time to explore all symbols indirectly connected to it
    • Click on a red socket to hide its symbol

  • Edit:
    • Click on a symbol to edit it
    • Shift-click on symbols, sockets or wires to select them
    • Press backspace to delete the selected objects
    • Double-click on the background of the visualizer to add a new symbol
