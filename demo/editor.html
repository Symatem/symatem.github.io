Show HN: Symatem meets WebAssembly - an online ontology engine, visualizer and editor

Symatem Ontology Editor

What you can see below is an ontology engine written in C++17, compiled to a binary WebAssembly file (88 KB) using Clang 4.0 and loaded into the browser. There some ES6 code, which handles the user interaction and the rendering, is connected to it via these two files: C++ side, JS side. The upper view contains a text representation of the ontology and the view below aggregates all triples sharing the same entity in boxes. In each box the common entity is displayed on top with a set of attribute (left) and value (right) pairs below. The blobs of these symbols are inlined graphically, so they don't have to be explored explicitly. 

(IsA Example; Quality Nice; Needs ToBeFound;)

Usage

  • Explore:
    • Drag and drop a symbol (box) around to reposition it
    • Click on a green or blue socket (circle) to toggle the visibility of the connected symbol
    • Click on a red socket to see all symbols indirectly connected to it

  • Edit:
    • Click on a symbol to edit it
    • Shift-click on symbols, sockets or wires to select them
    • Press backspace to delete the selected objects
    • Double-click on the background of the visualizer to add a new symbol
    • TODO: Drag a wire from a socket to add a new triple
    • Click on the source code above to edit it, blur again to apply the changes
